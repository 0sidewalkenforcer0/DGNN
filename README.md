## Important Parameters
`TEST_MODEL_PATH`: `None` for train [defaut], `Path` for test.  
`SAMPLER_W_QUALIFIERS`:  `True` for using the qualifiers [defaut].  
`SAMPLER_W_STATICS`: `True` for using the statics [defaut].
